# PHA Analysis Generator

A Next.js application for generating Process Hazard Analysis (PHA) reports quickly and efficiently.

## Features

- Landing page with hero section
- Product name and intended use input form
- Results table displaying hazard analysis
- Sign-in modal for accessing additional results

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/page.tsx` - Landing page
- `app/generate/page.tsx` - Form for product details
- `app/results/page.tsx` - Results table with hazards
- `components/SignInModal.tsx` - Sign-in modal component

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules
